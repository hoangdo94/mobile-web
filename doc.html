WEB MOBILE API

User

1 - Register
Definition: user register
Method: POST
URL: http://api.ws.hoangdo.info/users/
Data: {username, password, email}
Require: none
Response: status = 1 if success, 0 if error
2 - Get User Info
Definition: get user information (except password)
Method: GET
URL: http://api.ws.hoangdo.info/users/userId
Data: none
Require: userId
Response: status = 1 and user info if success, 0 if error
3 - Update User Info
Definition: update user information
Method: PUT
URL: http://api.ws.hoangdo.info/users/userId
Data: {username, password, email, name, avatar}
Require: userId, Authorization header
Response: status = 1 if success, 0 if error
4 - List all Users
Definition: List all Users
Method: GET
URL: http://api.ws.hoangdo.info/
Data: none
Require: Authorization header of admin account
Response: status = 1 and user list if success, 0 if error
5 - Delete user
Definition: Delete a user
Method: DELETE
URL: http://api.ws.hoangdo.info/users/userId
Data: none
Require: userId, Authorization header of admin account
Response: status = 1 if success, 0 if error

Authenticate

1 - Authenticate
Definition: user login
Method: POST
URL: http://api.ws.hoangdo.info/auth/
Data: {username, password}
Require: Authorization header
Response: status = 1 and corresponding user info if success, 0 if error

Book

1 - Get All Books
Definition: get all books
Method: GET
URL: http://api.ws.hoangdo.info/books/
Data: none
Require: none
Response: status = 1 and list of book if success, 0 if error
2 - Get A Book
Definition: get a book
Method: GET
URL: http://api.ws.hoangdo.info/books/bookId
Data: none
Require: bookId
Response: status = 1 and book info if success, 0 if error
3 - Add Book
Definition: add book
Method: POST
URL: http://api.ws.hoangdo.info/books/
Data: book info (title, author, cover, publishYear, genres, review)
Require: Authorization header
Response: status = 1 if success, 0 if error
4 - Update Book
Definition: update book information
Method: PUT
URL: http://api.ws.hoangdo.info/books/bookId
Data: book info (title, author, cover, publishYear, genres, review)
Require: bookId, Authorization header of the user who added the book, or admin
Response: status = 1 if success, 0 if error
5 - Delete Book
Definition: delete a book
Method: DELETE
URL: http://api.ws.hoangdo.info/books/bookId
Data: none
Require: Authorization header of the user who added the book, or admin
Response: status = 1 if success, 0 if error

Upload

1 - Upload Image
Definition: upload an image to server
Method: POST
URL: http://api.ws.hoangdo.info/upload
Data: form data
Require: dataType (json), mimeType (multipart/form-data), contentType (false), processData (false)
Response: status = 1 and url of image if success, 0 if error
